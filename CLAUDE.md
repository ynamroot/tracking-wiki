# 세션 시작 안내

이 저장소에서 작업하기 전에 순서대로 읽는다(`AGENTS.md`의 "시작 절차").

1. `PROJECT_CONTEXT.md`
2. 작업 대상 디렉터리의 가장 가까운 `AGENTS.md`
3. `wiki/index.md`, `wiki/log.md`

이 저장소는 공개다(GitHub public repo + Pages). 비공개 제품 초안, PRD, Feature Spec, Technical Spec의 내용은 어떤 형태로도 옮기지 않는다 — 경계는 `AGENTS.md`(공통 규칙, Agent 역할표)와 `OPERATIONS.md`("제품 초안 논증" 6절, "Productization" 8절, "Technical Review" 9절)를 따른다. 이 저장소를 근거로 삼는 비공개 작업 공간이 별도로 있으며, 그 경로는 이 저장소에 적지 않는다.
